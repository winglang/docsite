# ses

This library allows you to interact with the AWS SES Service

## Prerequisites

* [winglang](https://winglang.io).

## Installation

```sh
npm i @winglibs/ses
```

## Usage

```js
bring ses;

let emailService = new ses.EmailService(emailIdentities: ["bot@wing.cloud"]); 
```
